# 이벤트 뱅크 관리 문서

## 1. 버전 관리

### 1.1 버전 관리 정책
- 시맨틱 버저닝 적용 (MAJOR.MINOR.PATCH)
- 주요 변경사항은 MAJOR 버전 업
- 기능 추가는 MINOR 버전 업
- 버그 수정은 PATCH 버전 업

### 1.2 버전 관리 프로세스
1. 변경사항 식별
2. 버전 번호 결정
3. 변경 이력 기록
4. 배포 계획 수립
