# 이벤트 이미지 가이드라인

## 이미지 요구사항
- 파일 형식: PNG 또는 JPG/JPEG
- 해상도: 1920x1080px (16:9 비율)
- 최대 파일 크기: 2MB
- 색상 모드: RGB
- 이미지 품질: 최소 72dpi

## 파일명 규칙
- 이벤트 ID와 동일한 파일명 사용
- 소문자와 언더스코어(_) 사용
- 예시: cooking_time_pressure.png

## 폴더 구조
/images
  /daily_routine  - 일상 이벤트 이미지
  /crisis_events  - 위기 이벤트 이미지
  /opportunity    - 기회 이벤트 이미지

## 이미지 콘텐츠 가이드라인
1. 이미지는 이벤트의 상황을 명확하게 표현해야 함
2. 텍스트가 포함된 경우 가독성이 보장되어야 함
3. 브랜드 이미지와 일관성 유지
4. 저작권 문제가 없는 이미지 사용

## 임시 이미지 관리
- 현재 제공되는 이미지는 임시용입니다
- 실제 이미지로 교체 시 동일한 파일명으로 덮어쓰기
- 이미지 교체 후 캐시 초기화 필요 없음

## 주의사항
- 이미지 파일은 각 이벤트 카테고리에 맞는 폴더에 저장
- 이미지 교체 시 위의 규격을 반드시 준수
- 이미지 최적화를 통해 로딩 시간 최소화
